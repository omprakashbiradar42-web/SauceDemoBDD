# SauceDemo BDD Automation Framework

This project is an automated testing framework for the SauceDemo application using:

- Java
- Selenium
- Cucumber BDD
- Maven
- JUnit/TestNG (choose whichever you used)
- Page Object Model (POM)

## How to run the tests

1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Run the Runner class
4. Test reports will be generated in the `target` folder

## Project Structure

- `src/test/java` - Step definitions, feature files, runners
- `src/main/java` - Page classes and utilities
